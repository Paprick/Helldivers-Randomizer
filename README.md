# Helldivers Randomizer
Randomize your loadout for a mission in Helldivers
This includes:
- Randomize various weapons, stratagems, boosters and armour
- No repeat mode allowing to not roll items arleady used before
- Visual representation of no repeat mode items
- Stats and descriptions for weapons, stratagems, boosters and armour from game


## Purpose
To create cohesive experience between game a and a tool.

## Try it out
1. Clone the repository:
**git clone** https://github.com/Paprick/Helldivers-Randomizer.git

2. Navigate to the project directory:
**cd helldivers-randomizer**

3. Install dependencies:
**npm install**

4. Start the dev build:
**npm run dev**
